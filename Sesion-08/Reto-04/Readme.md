## Sesión 08 - Reto 04 

### OBJETIVO 
 - Darle la capacidad a tu App de hacer llamadas telefónicas por medio de Intents. 

#### REQUISITOS 
1. Android Studio
2. Emulador de Android o Dispositivo. 
3. Cable USB

#### DESARROLLO
En este reto vamos a llamar via teléfono. Este código puede funcionar con Apps que puedan llamar como Skype. Si bien es cierto que algunas Apps para hacer la marcación requieren permisos especiales, en el caso de Android cuando se usa el intent de la forma que marca el reto, puedes llamar a quien desees sin necesidad de ningún permiso especial.
