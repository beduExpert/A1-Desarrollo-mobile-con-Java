## Sesion02 - Reto 02

### OBJETIVO 
 - Aprender como cambiar el color de una aplicación.

#### REQUISITOS 
1. Android Studio
2. GIMP (opcional)


#### DESARROLLO
 Cambiar el color a los siguientes elementos.
* Colores de botones
* Colores de texto
* Colores de título
* Colores de fondo de pantalla

**Nota:**
 
Utilizar nombres descriptivos, como por ejemplo: 
**“color_dark_button”**
